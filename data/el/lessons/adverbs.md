# The Adverbs
Adverbs (Επιρρήματα) are indeclinable words that provide more information about verbs, adjectives, or other adverbs. They tell us **when**, **where**, **how**, or **how much** something happened.

::: content-grid mt-8

::: content-card

### Time (Χρονικά)

- σήμερα today
- αύριο tomorrow
- χτες yesterday
- τώρα now
- πάντα always
- ποτέ never

::: end

::: content-card

### Place (Τοπικά)

- εδώ here
- εκεί there
- πάνω up / on top
- κάτω down / below
- μέσα inside
- έξω outside

::: end

::: content-card

### Manner (Τροπικά)

- σιγά slowly / quietly
- γρήγορα quickly
- μαζί together
- έτσι like this
- ευτυχώς fortunately
- δυστυχώς unfortunately

::: end

::: content-card

### Quantity (Ποσοτικά)

- πολύ very / much
- λίγο a little
- αρκετά enough / quite
- καθόλου not at all

::: end

::: end

::: info-box mt-12

### Spelling Alert: πότε vs ποτέ

Be careful with the accent! **πότε** (accent on the first syllable) means "when?" **ποτέ** (accent on the last syllable) means "never".

::: end
