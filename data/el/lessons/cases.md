# The Cases
The role of a noun in a sentence is determined by its **case**. Modern Greek uses four primary cases.

::: content-grid

::: content-card
Ονομαστική
### Nominative

Role: **The Subject**

Question: **Who? / What?**

**Ο Νίκος** διαβάζει. **Nick** is reading.
::: end

::: content-card
Γενική
### Genitive

Role: **Possession**

Question: **Whose?**

Το χέρι **του Νίκου**. The hand **of Nick**.
::: end

::: content-card highlight
Αιτιατική
### Accusative

Role: **The Direct Object**

Question: **Whom? / What?**

Βλέπω **το(ν) Νίκο**. I see **Nick**.
::: end

::: content-card
Κλητική
### Vocative

Role: **Addressing**

Question: **Hey! Who?**

**Νίκο**, πού είσαι; **Nick**, where are you?
::: end

::: end

::: info-box mt-12

### Summary of Word Changes

Notice how the name **Νίκος** changed in each sentence:

- ο Νίκος (Nom)
- του Νίκου (Gen)
- τον Νίκο (Acc)
- Νίκο! (Voc)

::: end
