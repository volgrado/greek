# The Conjunctions
Conjunctions (Σύνδεσμοι) are words used to connect sentences, phrases, or individual words. They are essential for building complex thoughts and flowing conversation.

::: content-grid mt-8

### Connecting (Συμπλεκτικοί)

- και / κι and
- ούτε / μήτε neither / nor
- ή ... ή either ... or

### Contrasting (Αντιθετικοί)

- αλλά / μα but
- όμως / ωστόσο however / yet
- αν και although
- ενώ while

### Causal (Αιτιολογικοί)

- γιατί because / why
- επειδή since / as
- διότι because (formal)

### Time (Χρονικοί)

- όταν when
- πριν before
- αφού after / since
- μόλις as soon as

### Result (Συμπερασματικοί)

- άρα / επομένως therefore
- λοιπόν so / well
- ώστε so that

### Purpose / Condition

- να / για να to / so that
- αν / εάν if
- μήπως maybe / lest

::: end

::: example-box mt-12 mb-12

### Common Patterns

Θέλω **να** πάω **αλλά** δεν μπορώ. (I want **to** go **but** I cannot.)

Είναι καλό **γιατί** είναι φθηνό. (It is good **because** it is cheap.)

::: end
