# Demonstrative Pronouns
Demonstrative pronouns are used to point to specific objects or people, indicating their relative distance from the speaker.

::: content-grid mt-8

::: content-card
αυτός, αυτή, αυτό
### This (Close by)

Used for things near the speaker.

Αυτός είναι ο καφές μου. This is my coffee.
::: end

::: content-card
εκείνος, εκείνη, εκείνο
### That (Far away)

Used for things distant from the speaker.

Εκείνο το σπίτι. That house.
::: end

::: content-card
τέτοιος, τέτοια, τέτοιο
### Such

Used to indicate quality or type.

Δεν είναι τέτοιος άνθρωπος. He is not such a man.
::: end

::: content-card
τόσος, τόση, τόσο
### So much / many

Used to indicate quantity or degree.

Πήγαμε τόσο μακριά. We went so far.
::: end

::: end

::: info-box mt-12 mb-12

### Grammatical Note

When a demonstrative pronoun is used with a noun, the noun **must also have an article**. Example: **Αυτός ο** σκύλος (This [the] dog).

::: end
