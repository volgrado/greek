# Unit 27: Impersonal Verbs & Expressions

Sometimes a sentence doesn't have a specific "person" doing the action (e.g., "It is raining" or "It is necessary"). In Greek, we use **Impersonal Verbs** which are frozen in the **3rd Person Singular**.

---

## 1. The Power Pair: Πρέπει & Μπορεί
You’ve used these since A1, but now you understand *why* they work this way. They almost always trigger the Subjunctive (**να**).

- **Πρέπει να...** (It is necessary that... / **Must**)
- **Μπορεί να...** (It is possible that... / **Might**)

> [!WARNING]
> **Μπορώ vs Μπορεί**  
> - **Μπορώ** να πάω. (I can go—I have the ability).  
> - **Μπορεί** να πάω. (I might go—it is a possibility).

---

## 2. Essential Intermediate Verbs
These verbs describe allowed actions, appearances, or values. 

| Greek Verb | Meaning | Example |
| :--- | :--- | :--- |
| **Απαγορεύεται** | It is forbidden | **Απαγορεύεται** το κάπνισμα. |
| **Επιτρέπεται** | It is allowed | **Επιτρέπεται** να μπεις. |
| **Φαίνεται** | It seems | **Φαίνεται** ότι θα βρέξει. |
| **Αξίζει** | It is worth | **Αξίζει** να το δεις. |

---

## 3. Impersonal "Είναι" Expressions
You can turn any adjective into a command or observation by using **Είναι** (It is) + **να**.

- **Είναι δύσκολο να...** (It is difficult to...)
- **Είναι κρίμα να...** (It is a pity to...)
- **Είναι λογικό να...** (It makes sense to...)
