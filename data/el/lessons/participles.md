# Active & Passive Participles (Οι Μετοχές)

A Participle is a verb that has been dressed up to act like an adjective or an adverb. 

In English, we make Participles by adding **-ing** (Running water) or **-ed** (The closed door).

Greek does exactly the same thing. There are two types of Participles you need to master: **Active** and **Passive**.

---

## 1. The Active Participle (-οντας)

The Active Participle is used to show that two actions are happening *at the exact same time* by the same person. 

In English, we usually translate this as "**while [doing something]**" or "**by [doing something]**". 

To create the Active Participle, you take the Continuous (Present) stem of an active verb and add **-οντας** (or **-ώντας** for Type B verbs).

- τρέχω (I run) ➔ **τρέχοντας** (while running)
- διαβάζω (I read) ➔ **διαβάζοντας** (while reading)
- μιλάω (I speak - Type B) ➔ **μιλώντας** (while speaking)

> [!TIP]
> **The Golden Rule of the Active Participle:** 
> It NEVER changes its ending. It doesn't care about gender, number, or case. It is completely indeclinable. It always ends in an 's' sound!

Let's look at how it works in a sentence. Notice how it shows two things happening at once!

::: example-box 
**Ο Γιώργος τρώει βλέποντας τηλεόραση.**
(George eats *while watching* TV.)
::: end

::: example-box 
**Μαθαίνω ελληνικά ακούγοντας μουσική.**
(I learn Greek *by listening* to music.)
::: end

---

## 2. The Passive Participle (-μένος)

The Passive Participle is completely different. It acts exactly like a normal Adjective.

In English, this is the "**-ed**" or "**-en**" ending (The *broken* glass, the *hidden* door, the *tired* man). It describes the final state of an object *after* an action has been done to it.

To create the Passive Participle, you take the Passive stem and add **-μένος, -μένη, -μένο**.

Because it acts like an adjective, it is a chameleon! It must match the noun it describes in Gender, Number, and Case.

- κουράζω (I tire) ➔ **κουρασμένος** (tired)
- κλείνω (I close) ➔ **κλεισμένος** (closed)
- κρύβω (I hide) ➔ **κρυμμένος** (hidden) 

Look at how it behaves exactly like a normal adjective:

::: example-box 
**Η Μαρία είναι πολύ κουρασμένη σήμερα.**
(Maria is very *tired* today.)
*(Matches Feminine Singular: κουρασμένη).*
::: end

::: example-box 
**Βλέπω τις κλειδωμένες πόρτες.**
(I see the *locked* doors.)
*(Matches Feminine Plural Accusative: κλειδωμένες).*
::: end

---

## 3. The Ultimate Difference

It is crucial to understand the difference between the two participles. 

- **Active Participle (-οντας):** "While doing X". Describes an *ongoing action*. Never changes ending.
- **Passive Participle (-μένος):** "Having been X'd". Describes a *completed state*. Acts like an adjective and changes ending.

::: example-box 
**Κλαίγοντας** (while crying) vs **Κλαμένος** (tear-stained / having cried)
**Κρύβοντας** (while hiding something) vs **Κρυμμένος** (hidden)
::: end

- [ ] I know that the Active Participle is frozen and never changes its ending.
- [ ] I know that the Passive Participle (**-μένος**) translates to the "-ed" adjective form (tired, closed).
