# B2 Vocabulary: Adjectives & Comparisons

Words for evaluating the world around you and comparing things.

## 1. The Essential Evaluators
- **καλός, -ή, -ό** *(ka-lós)* = good ➔ **καλύτερος** (better)
- **κακός, -ή, -ό** *(ka-kós)* = bad ➔ **χειρότερος** (worse)
- **πολύς, πολλή, πολύ** *(po-lís)* = much / many ➔ **περισσότερος** (more)
- **λίγος, -η, -ο** *(lí-gos)* = little / few ➔ **λιγότερος** (less)

## 2. Describing Size & Shape
- **μεγάλος, -η, -ο** *(me-gá-los)* = big ➔ **μεγαλύτερος** (bigger / older)
- **μικρός, -ή, -ό** *(mik-rós)* = small ➔ **μικρότερος** (smaller / younger)
- **ψηλός, -ή, -ό** *(psee-lós)* = tall
- **κοντός, -ή, -ό** *(kon-dós)* = short
- **μακρύς, -ιά, -ύ** *(mak-rís)* = long

## 3. Describing Quality
- **νέος, -α, -ο / καινούριος** *(né-os)* = new / young
- **παλιός, -ά, -ό** *(pa-lyós)* = old (for things)
- **ωραίος, -α, -ο / όμορφος** *(o-ré-os)* = beautiful / nice
- **άσχημος, -η, -ο** *(ás-hee-mos)* = ugly

## 4. Comparison Keywords
- **πιο...** *(pyo)* = more... (the easy way: *πιο ωραίος*)
- **από...** *(a-pó)* = than... (*πιο ψηλός από εμένα*)
- **ο πιο...** *(o pyo)* = the most... (the superlative)
