# C1 Vocabulary: Common Compound Words

Greek loves to smash words together. Here are some of the most common compound words you will encounter every day, blending two concepts into a single word.

## 1. Time & Calendar Blends
- **το Σαββατοκύριακο** *(to sa-va-to-kí-rya-ko)* = the weekend (Saturday + Sunday)
- **το μερόνυχτο** *(to me-ró-neeh-to)* = a 24-hour period (day + night)
- **το ημερολόγιο** *(to ee-me-ro-ló-yee-o)* = the calendar / diary (day + word/logic)

## 2. Objects & Places
- **το μαχαιροπίρουνο** *(to ma-he-ro-pí-roo-no)* = cutlery (knife + fork)
- **το αλατοπίπερο** *(to a-la-to-pí-pe-ro)* = salt and pepper set
- **το βιβλιοπωλείο** *(to vee-vli-o-po-lí-o)* = the bookstore (book + to sell)
- **το ανθοπωλείο** *(to an-tho-po-lí-o)* = the florist (flower + to sell)
- **το ζαχαροπλαστείο** *(to za-ha-ro-plas-tí-o)* = the pastry shop (sugar + to mold)

## 3. Descriptive Compounds
- **ο καλοκάγαθος** *(o ka-lo-ká-ga-thos)* = good-hearted / kind (good + good)
- **ο στενοκέφαλος** *(o ste-no-ké-fa-los)* = narrow-minded / stubborn (narrow + head)
- **ο χαμογελαστός** *(o ha-mo-ye-las-tós)* = smiling (ground / low + laugh)
- **η καλημέρα** *(ee ka-lee-mé-ra)* = the good morning greeting (good + day)

## 4. Work & Society
- **η πολυκατοικία** *(ee po-lee-ka-tee-kí-a)* = apartment building (many + residence)
- **η τηλεόραση** *(ee tee-le-ó-ra-see)* = the television (far + vision)
- **το λεωφορείο** *(to le-o-fo-rí-o)* = the bus (people + carrier)
