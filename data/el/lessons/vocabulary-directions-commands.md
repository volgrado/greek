# B1 Vocabulary: Directions & Commands

Combine these words with the **Imperative Mood** and **Adverbs** to navigate the world.

## 1. Navigating the City (Αξιοθέατα)
- **ο δρόμος** *(o dró-mos)* = the road / street
- **το ξενοδοχείο** *(to xe-no-do-hí-o)* = the hotel
- **το φαρμακείο** *(to far-ma-kí-o)* = the pharmacy
- **το νοσοκομείο** *(to no-so-ko-mí-o)* = the hospital
- **το πάρκο** *(to pár-ko)* = the park
- **το αεροδρόμιο** *(to a-e-ro-dró-mee-o)* = the airport

## 2. Giving Directions (Imperatives)
- **Πήγαινε / Πηγαίνετε!** *(pí-ye-ne)* = Go! (Singular / Plural)
- **Στρίψε / Στρίψτε!** *(stríp-se)* = Turn!
- **Προχώρα / Προχωρήστε!** *(pro-hó-ra)* = Go ahead / Move forward!
- **Σταμάτα / Σταματήστε!** *(sta-má-ta)* = Stop!

## 3. Directional Adverbs
- **δεξιά** *(dex-yá)* = right
- **αριστερά** *(a-ris-te-rá)* = left
- **ευθεία** *(ef-thí-a)* = straight ahead
- **απέναντι** *(a-pé-nan-di)* = opposite / across
- **δίπλα σε** *(díp-la)* = next to
