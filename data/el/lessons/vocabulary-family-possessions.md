# B1 Vocabulary: Family Tree & Possessions

Perfect vocabulary for practicing the **Genitive Case** (e.g. "The brother *of the* mother").

## 1. Extended Family
- **ο αδερφός / ο αδελφός** *(o a-der-fós)* = the brother
- **η αδερφή / η αδελφή** *(ee a-der-fí)* = the sister
- **ο παππούς** *(o pa-poós)* = the grandfather (Genitive: *του παππού*)
- **η γιαγιά** *(ee ya-yá)* = the grandmother (Genitive: *της γιαγιάς*)
- **ο θείος** *(o thí-os)* = the uncle
- **η θεία** *(ee thí-a)* = the aunt
- **ο ξάδερφος / η ξαδέρφη** *(o xá-der-fos)* = the cousin

## 2. Belongings & Pets
- **το κλειδί** *(to klee-dí)* = the key (Plural: *τα κλειδιά*, Genitive: *του κλειδιού*)
- **το κινητό (τηλέφωνο)** *(to kee-nee-tó)* = the mobile phone
- **ο υπολογιστής** *(o ee-po-lo-yee-stís)* = the computer
- **ο σκύλος** *(o skí-los)* = the dog
- **η γάτα** *(ee gá-ta)* = the cat

## 3. Question Words for Genitive
- **Ποιανού είναι...;** *(pya-noo eé-ne)* = Whose is it? (Masculine / Neuter owner)
- **Ποιανής είναι...;** *(pya-nís eé-ne)* = Whose is it? (Feminine owner)
- **Τίνος είναι...;** *(tí-nos eé-ne)* = Whose is it? (Formal / Generic)
