# B2 Vocabulary: Expressions & Impersonal States

Verbs and nouns that are frozen in the 3rd person singular to describe states, rules, and the environment.

## 1. Impersonal Rule Verbs (The "It" Verbs)
- **πρέπει (να)** *(pré-pee)* = it is necessary (must)
- **μπορεί (να)** *(bo-reé)* = it is possible (might)
- **απαγορεύεται** *(a-pa-go-rév-e-te)* = it is forbidden
- **επιτρέπεται** *(e-pee-trép-e-te)* = it is allowed
- **αξίζει** *(ax-í-zee)* = it is worth it
- **φαίνεται** *(fén-e-te)* = it seems / it looks like

## 2. Weather & Nature (Τα Φαινόμενα)
*Weather is almost always impersonal in Greek.*
- **ο καιρός** *(o ke-rós)* = the weather
- **βρέχει** *(vré-hee)* = it is raining / **η βροχή** = the rain
- **χιονίζει** *(hyo-ní-zee)* = it is snowing / **το χιόνι** = the snow
- **φυσάει** *(fee-sá-ee)* = it is windy / **ο αέρας** = the wind
- **ο ήλιος** *(o eé-lyos)* = the sun
- **έχει ζέστη** *(é-hee zés-tee)* = it's hot (literally: "it has heat")
- **έχει κρύο** *(é-hee krí-o)* = it's cold

## 3. The "Είναι + να" Expressions
- **είναι κρίμα...** *(eé-ne krí-ma)* = it's a pity...
- **είναι λογικό...** *(eé-ne lo-yee-kó)* = it makes sense...
- **είναι δύσκολο / εύκολο...** *(dís-ko-lo / éf-ko-lo)* = it is difficult / easy...
