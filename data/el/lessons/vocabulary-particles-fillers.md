# B2 Vocabulary: Conversational Fillers & Particles

The little words that make you sound like a native.

## 1. The Greek "Um" and "Well"
- **λοιπόν** *(lee-pón)* = well... / so...
- **δηλαδή** *(dee-la-dée)* = meaning... / that is (to say)...
- **βασικά** *(va-see-ká)* = basically / actually
- **ούτως ή άλλως** *(oó-tos ee ál-los)* = anyway / either way

## 2. Expressing Doubt & Sarcasm
- **άραγε** *(á-ra-ge)* = I wonder... (used in questions)
- **δήθεν / τάχα** *(dee-then / tá-ha)* = supposedly / "yeah, right"
- **μάλλον** *(mál-lon)* = probably / I guess
- **ίσως** *(í-sos)* = perhaps / maybe

## 3. Strong Emotion & Wishes
- **μακάρι** *(ma-ká-ree)* = I hope / if only!
- **ας** *(as)* = let's / let it be (*ας πάμε* = let's go)
- **άντε / άιντε** *(án-de)* = come on! / hurry up!
- **αλίμονο** *(a-lí-mo-no)* = alas / of course!

## 4. Agreement & Disagreement
- **ακριβώς!** *(a-kree-vós)* = exactly!
- **φυσικά / βεβαίως** *(fee-see-ká / ve-vé-os)* = naturally / certainly
- **αποκλείεται!** *(a-po-klí-e-te)* = no way! / impossible!
