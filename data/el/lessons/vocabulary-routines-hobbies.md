# A2 Vocabulary: Routines & B-Group Verbs

Common Type B verbs (B1 & B2 "singers") and expressions used to describe habits and routines.

## 1. Daily Routine Verbs (B1 -άω)
These verbs love the *Alpha* sound. Remember: use them for habits (Imperfect/Continuous)!

- **ξυπνάω** *(xip-ná-o)* = to wake up
- **μιλάω** *(mee-lá-o)* = to speak / talk
- **φοράω** *(fo-rá-o)* = to wear (clothes)
- **πεινάω** *(pee-ná-o)* = to be hungry
- **διψάω** *(deep-sá-o)* = to be thirsty
- **αγαπάω** *(a-ga-pá-o)* = to love

## 2. Essential B2 Verbs (-ώ)
These are crucial verbs for requesting and giving permission.

- **μπορώ** *(bo-ró)* = to be able to / can
- **ζω** *(zo)* = to live
- **τηλεφωνώ** *(tee-le-fo-nó)* = to call on the phone
- **λειτουργώ** *(lee-toor-gó)* = to function / operate

## 3. Routine Nouns
Combine these with verbs to describe your day.

- **το πρωί** *(to pro-í)* = the morning
- **το μεσημέρι** *(to me-see-mé-ree)* = noon / midday
- **το απόγευμα** *(to a-pó-yev-ma)* = the afternoon
- **το βράδυ** *(to vrá-dee)* = the evening / night
- **η δουλειά** *(ee doo-lyá)* = work / job
