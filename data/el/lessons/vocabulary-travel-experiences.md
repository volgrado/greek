# B1 Vocabulary: Travel & Life Experiences

These words are naturally used with the **Perfect Tenses** (Have you ever...?) and **Conditionals** (If I go...).

## 1. Life Experiences Verbs
- **ταξιδεύω** *(ta-xi-dév-o)* = to travel (Perfect: *έχω ταξιδέψει*)
- **επισκέπτομαι** *(e-pis-kép-to-me)* = to visit (Perfect: *έχω επισκεφτεί*)
- **γνωρίζω** *(gno-rí-zo)* = to meet / to know (Perfect: *έχω γνωρίσει*)
- **δοκιμάζω** *(do-kee-má-zo)* = to try / to taste (Perfect: *έχω δοκιμάσει*)

## 2. Travel Nouns
- **το ταξίδι** *(to ta-xí-di)* = the trip / journey
- **η χώρα** *(ee hó-ra)* = the country
- **το εισιτήριο** *(to ee-see-tí-ree-o)* = the ticket
- **οι διακοπές** *(ee dya-ko-pés)* = the holidays (always plural!)
- **το νησί** *(to nee-sí)* = the island
- **η θάλασσα** *(ee thá-las-sa)* = the sea

## 3. The "Experience" Adverbs
- **όταν** *(ó-tan)* = when (not for questions; used for "when I was there...")
- **αν** *(an)* = if (the key to Conditionals!)
- **ήδη** *(í-dee)* = already (*Το έχω ήδη κάνει* - I have already done it)
- **ακόμα** *(a-kó-ma)* = yet / still (*Δεν το έχω δει ακόμα* - I haven't seen it yet)
