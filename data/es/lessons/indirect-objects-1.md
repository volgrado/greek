# Indirect Object Pronouns I
An indirect object tells us **to whom** or **for whom** an action is performed. Indirect object pronouns replace these nouns.

## The Forms

| Person | Singular | Plural |
|---|---|---|
| 1st | me (to/for me) | nos (to/for us) |
| 2nd | te (to/for you) | os (to/for you all) |
| 3rd | le (to/for him/her/you) | les (to/for them/you all) |

::: info-box

### Placement

Similar to direct object pronouns, indirect ones usually come immediately **before** the conjugated verb.

Juan **me** da el libro.

Juan gives the book to me.

::: end
