# Indirect Object Pronouns III
Just like direct object pronouns, indirect pronouns can also be attached to infinitives and present participles.

## Attachment Examples

::: example-box
With Infinitive: Quiero dar**le** el regalo.

With Participle: Estoy dándo**le** el regalo.

::: end

::: info-box

### Remember the Choice!

You always have the alternative option of placing the pronoun before the conjugated verb:

Le quiero dar el regalo.

::: end
