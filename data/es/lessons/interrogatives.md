# Interrogatives
Spanish uses unique punctuation and specific question words to ask questions.

## Punctuation Rule

In Spanish, questions always start with an inverted question mark (**¿**) and end with a standard one (**?**).

## Common Question Words

| Spanish | English |
|---|---|
| ¿Qué? | What? |
| ¿Quién? | Who? |
| ¿Cuándo? | When? |
| ¿Dónde? | Where? |
| ¿Por qué? | Why? |
| ¿Cómo? | How? |

### Accent Marks

Notice that all question words in Spanish have an accent mark when used in a question.
