# Negative Commands
To tell someone NOT to do something, the rules change significantly, especially for the 'tú' form.

## The Rule for All Forms

To form any negative command (Ud., Uds., or tú), start with the **formal command** form and add 'no' in front.

- Usted: **No hable** (Don't speak)
- Ustedes: **No hablen** (Don't speak)

## Negative Tú Commands

For the 'tú' form, take the Usted command form and add **-s**.

- Hablar → No hable → **No hables**
- Comer → No coma → **No comas**
- Decir → No diga → **No digas**
