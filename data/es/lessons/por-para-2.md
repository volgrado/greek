# Por vs Para II
While 'por' is about the 'through' or the 'why,' **para** is about the 'for' in terms of destination, goal, or deadline.

## Primary Uses of PARA

Think of **para** as expressing a destination, a purpose, or a point in the future.

- **Destination** (toward, in the direction of)
- **Deadline or specific time** (by, for)
- **Purpose or goal** (in order to + infinitive)
- **Recipient** (for)

## Examples of Para

- Salgo para Madrid. (I'm leaving for Madrid (Destination).)
- Esta tarea es para mañana. (This homework is for tomorrow (Deadline).)
- Estudio para aprender. (I study in order to learn (Purpose).)
- Este regalo es para ti. (This gift is for you (Recipient).)
