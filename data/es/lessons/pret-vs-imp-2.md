# Preterite vs Imperfect II
Often, both tenses appear in the same sentence to describe an action that was in progress when something else happened.

## Interrupting Actions

The **imperfect** provides the background/setting, and the **preterite** is the interruption.

::: example-box
Yo **leía** (Imp) cuando Juan **entró** (Pret).

I was reading when Juan entered.

::: end

## Simultaneous Actions

If two actions were happening at the same time, use the **imperfect** for both.

Ella **cantaba** mientras yo **bailaba**.
