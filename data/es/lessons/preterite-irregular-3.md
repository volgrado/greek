# Preterite: Irregular Verbs III
Continuing with the irregular stems that share the same endings (-e, -iste, -o, -imos, -ieron).

## Verbs: Poder, Poner, Saber

- Poder → **pud-** (to be able to)
- Poner → **pus-** (to put)
- Saber → **sup-** (to know facts)

## Example Conjugations

::: example-box
Poder: pude, pudiste, pudo, pudimos, pudieron

Poner: puse, pusiste, puso, pusimos, pusieron

Saber: supe, supiste, supo, supimos, supieron

::: end

::: info-box

### Meaning Changes

In the preterite, some of these verbs take on slightly different meanings:

- Saber (preterite) = to find out
- Poder (preterite) = to manage to/succeed

::: end
