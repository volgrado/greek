# Preterite: Regular Verbs
To conjugate regular verbs in the preterite, remove the -ar, -er, or -ir ending and add these forms.

## -ar Endings

| yo | -é |
|---|---|
| tú | -aste |
| él | -ó |
| nos. | -amos |
| ellos | -aron |

## -er / -ir Endings

Notice that -er and -ir verbs share the same set of preterite endings!

| yo | -í |
|---|---|
| tú | -iste |
| él | -ió |
| nos. | -imos |
| ellos | -ieron |
