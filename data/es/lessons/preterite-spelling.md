# Preterite: Spelling Changes
To preserve the original pronunciation of the infinitive, some verbs require a spelling change in the **yo** form only.

## The -car, -gar, -zar Rule

- -car → **-qué** (buscar → busqué)
- -gar → **-gué** (pagar → pagué)
- -zar → **-cé** (empezar → empecé)

### Note

These changes only happen in the first-person singular. All other forms follow the regular -ar pattern.
