# Combined Object Pronouns
When a sentence has both a direct and an indirect object pronoun, follow these two critical rules.

## Rule 1: Order

The indirect object pronoun always comes **before** the direct object pronoun.

Mnemonic: **RID** (Reflexive, Indirect, Direct).

Él **me lo** da.

He gives it to me. (me = indirect, lo = direct)

## Rule 2: The 'Se' Rule

If both pronouns start with an 'l' (le/les + lo/la/los/las), the indirect pronoun changes to **se** to avoid an awkward sound.

Incorrect: Le lo doy. → Correct: **Se lo** doy.
