# Regular Verbs II (-er)
Regular -er verbs follow a pattern similar to -ar verbs, but using 'e' instead of 'a' in the endings.

## Conjugation Pattern

To conjugate a regular -er verb in the present tense, remove the -er to find the stem, then add the following endings:

| Subject | Ending | Example (Comer) |
|---|---|---|
| Yo | **-o** | como |
| Tú | **-es** | comes |
| Él / Ella / Ud. | **-e** | come |
| Nosotros/as | **-emos** | comemos |
| Vosotros/as | **-éis** | coméis |
| Ellos / Ellas / Uds. | **-en** | comen |
