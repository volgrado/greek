# Accidental 'Se'
Spanish uses a special construction with 'se' to describe accidental or unintentional events, shifting the blame away from the person.

## The Construction

Pattern: **SE + Indirect Object Pronoun + Verb (+ Subject)**

- Se me rompió el vaso. (The glass broke [on me / accidentally].)
- Se nos olvidaron las llaves. (We forgot the keys [accidentally].)

::: info-box

### Grammar Logic

In this structure, the thing that was dropped/broken is the **subject**, and the person is merely the indirect object affected by the event.

::: end
