# Ser vs Estar I (Conjugation)
In English, the verb 'to be' is always the same. In Spanish, there are two distinct verbs: **ser** and **estar**.

## Conjugation of Ser

| Subject | Ser |
|---|---|
| Yo | soy |
| Tú | eres |
| Él / Ella / Ud. | es |
| Nosotros/as | somos |
| Vosotros/as | sois |
| Ellos / Ellas / Uds. | son |

## Conjugation of Estar

| Subject | Estar |
|---|---|
| Yo | estoy |
| Tú | estás |
| Él / Ella / Ud. | está |
| Nosotros/as | estamos |
| Vosotros/as | estáis |
| Ellos / Ellas / Uds. | están |
