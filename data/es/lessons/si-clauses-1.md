# Si Clauses I
'Si' clauses (If clauses) express conditions. There are several types depending on how likely the condition is to be true.

## Type 1: Possible Conditions

If the condition is likely or possible, use the **Indicative**.

Pattern: **Si + Present Indicative → Present/Future Indicative**

::: example-box
Si tengo dinero, iré a España. (If I have money, I will go to Spain.)

Note: NEVER use the present subjunctive after 'si' to express a possible condition.

::: end
