# Stem-Changers: o -> ue
Just like e->ie verbs, these verbs change the 'o' in the stem to 'ue' for all forms except nosotros/vosotros.

## Example: Dormir (to sleep)

| Subject | Conjugation |
|---|---|
| Yo | d**ue**rmo |
| Él / Ella | d**ue**rme |
| Nosotros | dormimos |
| Ellos | d**ue**rmen |

- Poder → Puedo ((I can))
- Almorzar → Almuerzo ((I eat lunch))
