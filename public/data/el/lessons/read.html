Learn to Read

Reading Greek is a process of assembling phonetic blocks (syllables). Unlike English, Greek is highly phonetic—once you know the sound of a letter or a "double letter", it almost always sounds the same.

Syllabic Assembly

Follow these examples to see how sounds combine to form words.

::: content-card μα + μάμαμά mama / mother ::: end

::: content-card πα + τέ + ραςπατέρας pateras / father ::: end

::: content-card ου + ρα + νόςουρανός ouranos / sky ::: end

::: content-card τζά + κιτζάκι tzaki / fireplace ::: end

Key Takeaway

Treat every word as a series of distinct sounds. Notice how the accent mark dictates where the voice rises. Without the accent mark, we wouldn't know which syllable to emphasize!
