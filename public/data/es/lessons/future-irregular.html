Future Tense: Irregular

Even irregular future verbs use the standard endings. The only thing that changes is the stem of the verb.

Common Irregular Stems

  • Tener → tendr-
  • Poner → pondr-
  • Salir → saldr-
  • Venir → vendr-
  • Poder → podr-
  • Saber → sabr-
  • Hacer → har-
  • Decir → dir-
  • Querer → querr-

Example: Tener

tendré, tendrás, tendrá, tendremos, tendrán

Note

Irregular verbs in the future will almost always be irregular in the conditional tense as well, using the same stems.
