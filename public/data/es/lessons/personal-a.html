The Personal 'a'

When the direct object of a verb is a person, Spanish requires a special preposition: the personal 'a'.

Usage

It doesn't translate to English, but it must be there if the object is human or a pet.

Object is a thing: Veo el libro.

Object is a person: Veo a Juan.

  • Llamo a mi madre. (I call my mother.)
  • Conozco a esa señora. (I know that lady.)
